# LinkPedia 🔗

Projeto desenvolvido com Django, com foco em praticar operações CRUD, autenticação de usuários e conceitos de TDD.

---

#  Sobre o projeto

O **LinkPedia** foi criado para um cliente fictício chamado **Orlando Saraiva Jr** (meu professor), que precisava de uma plataforma simples para organizar links importantes de estudos, pesquisas e materiais acadêmicos.

A ideia do sistema é permitir que usuários possam:

- ✅ Cadastrar links
- ✅ Atualizar links
- ✅ Excluir links
- ✅ Visualizar links cadastrados
- ✅ Fazer login para acessar o sistema
- ✅ Receber mensagens de sucesso nas operações

O projeto foi desenvolvido com foco em aprendizado, organização de código e boas práticas básicas utilizando Django.

---

# 🛠️ Tecnologias utilizadas

- Python
- Django
- Bootstrap 5
- HTML5
- CSS3
- SQLite3

#  Testes automatizados

O projeto possui testes automatizados utilizando o sistema de testes do Django.

Foram realizados testes para:

- Templates

- Seleção de links
- Autenticação
